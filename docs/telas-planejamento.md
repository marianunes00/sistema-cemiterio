# Telas do Sistema

## Tela de Menu Principal/Todos os tipos de usuário
(A tela de menu principal apresenta um menu lateral que reúne as principais funcionalidades do sistema. Nesse menu, o usuário pode acessar as opções de sepultura,falecido e serviços.
No lado direito da tela é exibida a área principal, contendo o logotipo do sistema e uma representação do mapa do cemitério, servindo como painel inicial de navegação.)

## Tela de Cadastro de Sepultura/Usuário administrador
(A tela de cadastro de sepultura apresenta um formulário onde o usuário preenche informações como número da sepultura, localização (lote), tipo, status e data de criação,ao final do formulário se encontra o botão cadastrar,e botões  adicionais de atualizar e excluir que funcionarão quando uma sepultura for selecioonada na tabela que é exibida abaixo do formulário.)

## Tela de Cadastro de Sepultura/Usuário atendente e manutenção
(A tela de cadastro de sepultura mostra uma tabela com as informações de cada sepultura)

## Tela de Cadastro de Falecido/Usuário administrador
(Nesta tela é exibido um formulário de dados pessoais do falecido, incluindo informações como identificação, certidão de óbito, data de falecimento e a sepultura associada,ao final do forulário haverá o botão cadastrar,e botões  adicionais de atualizar e excluir que funcionarão quando um falecido for selecioonado na tabela que é exibida abaixo do formulário.)

## Tela de Cadastro de Falecido/Usuário atendente e manutenção
(Nesta tela é exibido uma tabela com dados de cada falecido cadastrado no Sistema.)

## Tela de Agenda de Serviços/Usuário administrador
(A tela de serviços apresenta um formulário com campos como id do serviço,tipo,data,status e notificação  botão cadastrar,ao final do formulário haverá o botão cadastrar e botões  adicionais de atualizar e excluir, que funcionarão quando um  serviço for selecioonado na tabela que é exibida abaixo do formulário.)

## Tela de Agenda de Serviços/Usuário administrador
(A tela de serviços apresenta um A tela exibe um formulário simplificado com campos como status e notificação internas,ao final há o botão de atualizar que pode ser utilizado quando um  serviço for selecioonado na tabela que é exibida abaixo do formulário.)

## Tela de Agenda de Serviços/Usuário atendente
(A tela de serviços apresenta uma tabela com as informações dos serviços cadastrados.)

## Tela de Relatórios/Todos os tipos de usuário
(Ao selecionar a opção de relatórios, o usuário é direcionado para esta tela, onde pode escolher entre relatório de falecidos, relatório de sepulturas ou relatório combinado.
Após selecionar o tipo desejado, o usuário deve clicar no botão Gerar, e os dados serão exibidos de acordo com os filtros escolhidos.)

## Tela de avisos/Todos os tipos de usuário
(Nesta tela terá uma lista de avisos e notificações relacionados aos serviços internos do cemitério.)
