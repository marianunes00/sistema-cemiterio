# Telas do Sistema

## Tela de Menu Principal
(A tela de menu principal apresenta um menu lateral que reúne as principais funcionalidades do sistema. Nesse menu, o usuário pode acessar as telas de cadastro (sepultura e falecido), visualização (lista de falecidos cadastrados, organizada alfabeticamente, e o mapa de sepulturas), além da seção de serviços.
No lado direito da tela é exibida a área principal, contendo o logotipo do sistema e uma representação do mapa do cemitério, servindo como painel inicial de navegação.)

## Tela de Cadastro de Sepultura
(A tela de cadastro de sepultura apresenta um formulário onde o usuário preenche informações como número da sepultura, localização (numero, lote), tipo, status e data de criação.
Ao final da tela, há os botões Salvar, Limpar e Cadastrar Falecido, sendo este último responsável por abrir o formulário de cadastro de falecido já vinculado à sepultura recém-inserida.)

## Tela de Cadastro de Falecido
(Nesta tela é exibido um formulário de dados pessoais do falecido, incluindo informações como identificação, certidão de óbito, data de falecimento e a sepultura associada.
No rodapé do formulário, estão disponíveis os botões Salvar e Limpar, permitindo registrar ou reiniciar o preenchimento dos dados.)

## Tela de Agenda de Serviços
(A tela de serviços apresenta opções como atualizar e excluir dados(sepultura,falecido) esse só para usuário administrador, gerar relatórios, exumação e translados, limpeza e manutenção. Cada botão leva o usuário a uma ação correspondente, permitindo o agendamento, visualização ou atualização dos serviços realizados no cemitério e dados já cadastrados.)

## Tela de Relatórios
(Ao selecionar a opção de gerar relatórios, o usuário é direcionado para esta tela, onde pode escolher entre relatório de falecidos, relatório de sepulturas ou relatório combinado.
Após selecionar o tipo desejado, o usuário deve clicar no botão Gerar, e os dados serão exibidos de acordo com os filtros escolhidos.)
